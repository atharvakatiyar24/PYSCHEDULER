# PYSCHEDULER
A lightweight, terminal-based calendar and task management application built with Python. It allows users to view specific calendar months, check for existing tasks, and schedule new plans for specific dates.

🚀 Features:
* View Calendar: Display any month and year using Python's built-in calendar module.

* Manage Tasks: Check if a task exists for a specific date, or add a new one instantly.

* View All Tasks: Display a complete summary of all your scheduled tasks.

* Simple Interactive Menu: Easy-to-use command-line interface (CLI).

🛠️ Built With:
*Python 3

*calendar module (Standard Library)

📦 How to Run:
1. Clone this repository:
   

git clone https://github.com/yourusername/PyScheduler.git

3. Navigate into the project folder:
   

cd PyScheduler

5. Run the application:
   

python main.py
